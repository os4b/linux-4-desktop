Installer Linux sur un ordinateur
1) Téléchargement de l'iso Mint 22.2 sur https://linuxmint.com/
2) Téléchargement de l'iso Windows 10 sur https://www.microsoft.com/fr-fr/software-download/
3) création d'un support d'instalation avec Multi OS USB
4) Démarer sur Linux Console avec la clef USB
5) Création de deux partitions de la moitié du disque
6) Copie des images ave cdmesg cf image : dmesg.jpg
7) booter sur la clef
8) lancer le programme d'instalation de windows cf image : windowsInstallationConfiguration.png
9) installation de Windows sur la première partition du disque
10) une fois terminé, rebooter sur la clef
11) lancer le programme d'instalation de Linux Mint
12) Installer Mint sur la deuxiemme partition du disque cf image : mintInstallationConfiguration.jpg
